# 配件资料 - 产品需求文档 (PRD)

> **系统：** 车配智数供应链平台 (gyl.lxcf.cn)
> **路径：** 商品/仓库 → 商品管理 → 配件资料
> **版本：** v1.1
> **日期：** 2026-05-25

---

## 1. 概述

### 1.1 功能定位
配件资料是商品管理的核心模块，用于维护平台所有配件商品的基础档案、价格策略、图片详情和标签归类信息，支持单个新增/编辑、批量操作、模板导入导出和数据审核。

### 1.2 用户角色
| 角色 | 权限 |
|------|------|
| 超级管理员 | 全部权限（增删改查、导入导出、审核） |
| 产品经理 | 新增/编辑配件、管理归属 |
| 仓库管理员 | 查询、查看配件信息 |

## 8. 交互说明

### 8.1 删除确认
- 点击表格行内「删除」按钮
- 弹出确认框：「此操作将永久删除该数据, 是否继续?」
- **[取消]** 关闭弹框
- **[确定]** 执行删除

### 8.2 批量操作
- 批量匹配车型、批量编辑、批量删除均需先勾选数据行
- 未勾选时点击「批量删除」，按钮显示为禁用状态「请先选择要删除的数据」
- 勾选后进入批量操作模式，显示「退出批量删除」

### 8.3 导入流程
1. 下载模板：[模版下载]
2. 按模板格式填写配件数据
3. 点击[导入]按钮，打开文件选择器，选择文件
4. 系统弹出「配件导入」弹窗，显示导入预览/确认信息
5. 数据进入审核流程（在导入记录Tab查看）

### 8.4 导出
- 点击[导出]按钮，弹出「导出栏目设置」弹窗
- 弹窗提示：「导出的字段默认与原先保持一致，可调整勾选、顺序和表头名称后再导出」
- 可勾选/取消字段，调整导出顺序，修改表头名称
- 点击弹窗内[导出]按钮，导出当前筛选条件下的数据为Excel文件
